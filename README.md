## sociable-lapwing-publication
- This repository serves as the data and analysis portal for the sociable lapwing paper published in the __Journal of Ornithology__.
- Files here will feed into and mirror the DRYAD repository shared to the journal.
- They will mostly likely not updated anymore given the work is complete, but repository should remain functional for data reproducibility and sharing.
- Any future work, or funding can potentially benefit from this as well
